üìñ Using Docker

Prefer Docker? We've got you covered:

With Docker Compose (Easiest):

# Create .env file with your credentials echo "SNIPPET_PASSWORD=your-secure-password" > .env # Start the app docker-compose up -d # View logs docker-compose logs -f

With Docker only:

# Build docker build -t snippets-app . # Run docker run -d -p 5000:5000 \ -e SECRET_KEY="your-secret-key" \ -e SNIPPET_PASSWORD="your-password" \ --name snippets snippets-app

üîë Security Tips

  • Always use a strong, unique password
  • Generate a random SECRET_KEY: python -c "import os; print(os.urandom(24).hex())"
  • Use HTTPS in production (most platforms provide this automatically)
  • Don't share your credentials or commit them to version control
  • Backup your snippets.db file regularly
